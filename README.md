Opening the Menu

Press P to open the Gamble Rig menu.

Before using the mod, go into Minecraft → Options → Controls and make sure P is not already bound to something else. If another action is using P, unbind it or change it, otherwise the menu may not open.

Setting Up the Rig

The rig uses two dispensers and a set of selectable items.

Open the menu with P.

Hold the item you want to assign in your hand. Click P and Click Assign Hand

Repeat for the second item with the other item!.

Make sure you number all the items 1-9 only numbers nothing else on it!
